3.2. Creating a Meeting

If you have the correct access rights within a Category you can create a new meeting using the 'Meeting' option in the Tools menu, you will be presented with a form similar to the one you would for a Conference, however you are able to choose the default style in which the Meeting will be displayed (See Meeting Display) when you submit this form the Meeting will be created.
