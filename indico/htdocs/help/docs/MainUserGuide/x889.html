6.3. Sharepoint

If you maintain a sharepoint web site, it is very easy to create inside it a web part exposing the forthcoming events from an Indico category. First add an XML web part, then in the "XML link" part, add the XML export URL from indico (eg. http://indico.cern.ch/tools/export.py?fid=2l12&date=today&days=1000&of=xml) and in the "XSL link" part, add this URL: http://indico.cern.ch/export.xsl.
