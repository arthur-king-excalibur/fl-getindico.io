4.2. Creating a Lecture

If you have the correct access rights within a Category you can create a new lecture using the 'Lecture' option in the Tools menu, you will be presented with a form similar to the one you would for a Meeting, you are able to choose the default style in which the Lecture will be displayed (See Lecture Display) when you submit this form the Lecture will be created.
